# 技术栈
* springboot 3.2.3
* jdk 17
# 功能
* 支持自定义表格标题
* 支持自定义标题样式
* 支持自定义列样式
* 支持自定义行样式
* 支持自定义文件名
# 快速开始
* maven
```java
<dependency>
    <groupId>io.github.collectbugs</groupId>
    <artifactId>easyhtml</artifactId>
    <version>1.0.1</version>
</dependency>
```
